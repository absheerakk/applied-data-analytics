# Applied Data Analytics

A collection of Python implementations covering fundamental concepts in data analytics, machine learning, natural language processing (NLP), clustering, and data mining.

This repository contains practical programs developed for learning, experimentation, and understanding core analytical techniques using Python and popular machine learning libraries.


## Topics Covered

### Natural Language Processing (NLP)
- Bag of Words (BoW) Vectorization
- TF-IDF Document Ranking
- Cosine Similarity
- Naive Bayes Text Classification

### Machine Learning
- Decision Tree Rule Extraction
- Information Gain Split Selection
- Multilayer Perceptron (MLP) for XOR Classification

### Clustering & Similarity Analysis
- PAM (Partitioning Around Medoids) Clustering
- Distance Metrics:
  - Euclidean Distance
  - Manhattan Distance
  - Minkowski Distance

### Data Mining
- Apriori Association Rule Mining
- Frequent Itemset Generation
